To start the Backend:

-Open terminal in project's root directory (Tranzact-Challenge-Node.JS)
-Execute command cd Backend
-Execute command node index.js

To start the Frontend:

-Open the project's root directory
-Open Frontend
-Open index.html in browser

Considerations:

-Default language is english
-If you have spanish as your default browser language,
the text will display in spanish
-If you have any other language as your default browser
language, the text will display in english
